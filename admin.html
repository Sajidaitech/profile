How to Get a GitHub Token

  1. Go to github.com → click your profile photo → Settings

  2. Scroll down to Developer settings → Personal access tokens → Tokens (classic)

  3. Click Generate new token (classic). Give it a name like portfolio-admin.

  4. Tick the repo checkbox. Set expiry as needed. Click Generate token.

  5. Copy the token (starts with ghp_) and paste it above. It is stored only in your browser session.
